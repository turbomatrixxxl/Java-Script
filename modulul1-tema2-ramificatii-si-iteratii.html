Iterații

O sarcină des întâlnită în programare este să efectuăm același tip de acțiune de mai multe ori. De exemplu, să eliminăm clienții din listă, unul câte unul, sau să verificăm salariile angajaților și să executăm același cod pentru fiecare dintre ei. În astfel de scopuri, pentru repetarea continuă a unei secțiuni de cod - sunt folosite buclele(loops).

  • Loop(buclă) - o structură de control din limbajele de programare high level, concepută pentru a organiza execuția multiplă a unui set de instrucțiuni.
  • Loop body(corpul buclei) - o secvență de instrucțiuni destinate executării multiple.
  • Iterație - execuție unică a întregului cod din corpul buclei.
  • Counter - o variabilă care stochează numărul iterației curente. Bucla nu conține neapărat un numărător și nici nu trebuie să fie unul, condiția de ieșire din buclă poate depinde de mai multe variabile care sunt modificate în buclă în timpul execuției.

Ciclul while

while (condition) { // cod, corpul buclei (statement) }

Ciclul do...while

do { // statement } while (condition);

Ciclul for

for (initialization; condition; post-expression) { // statements }

Operatorul break

Putem întrerupe bucla în orice moment. Pentru a face acest lucru, există o instrucțiune break, care oprește complet execuția buclei și continuă execuția codului dinafara buclei.

Operatorul continue

Nu întrerupe întreaga buclă, ci doar execuția iterației curente. Este folosit doar dacă nu există nimic de făcut la iterația curentă a buclei și este timpul să trecem la următoarea iterație.
